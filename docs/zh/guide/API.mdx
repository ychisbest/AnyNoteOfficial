# API 文档

## 获取所有笔记

- **URL**: `https://your_server/api/notes`
- **方法**: `GET`
- **Header**:
  - `x-secret`: `your secret`
- **响应**:
  ```json
  [
    {
      "id": 0,
      "isTopMost": true,
      "content": "string",
      "createTime": "2024-09-19T07:44:53.015Z",
      "lastUpdateTime": "2024-09-19T07:44:53.015Z",
      "archiveTime": "2024-09-19T07:44:53.015Z",
      "isArchived": true,
      "color": 0,
      "index": 0
    }
  ]
  ```

## 创建笔记

- **URL**: `https://your_server`
- **方法**: `POST`
- **Header**:
  - `x-secret`: `your secret`
- **请求体**:
  ```json
  {
    "content": "content"
  }
  ```
- **响应**:
  ```json
  {
    "id": 0,
    "isTopMost": true,
    "content": "string",
    "createTime": "2024-09-19T07:44:53.015Z",
    "lastUpdateTime": "2024-09-19T07:44:53.015Z",
    "archiveTime": "2024-09-19T07:44:53.015Z",
    "isArchived": true,
    "color": 0,
    "index": 0
  }
  ```
