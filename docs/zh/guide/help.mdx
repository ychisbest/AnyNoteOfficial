# 操作说明

## 添加笔记
- **操作**：点击界面上的加号按钮。
- **功能**：打开一个新的笔记编辑窗口，准备输入内容。
<img src="/screenshot/addnote.gif" alt="loginpage" style={{ width: '350px', height: 'auto',boxShadow: '0 0 10px 0 rgba(0, 0, 0, 0.1)' }} />

## 编辑笔记
- **操作**：在笔记编辑窗口中实时输入内容。
- **功能**：支持Markdown语法，例如标题、列表、CheckList等。
- **同步状态**：右上角显示当前笔记的同步状态。
<img src="/screenshot/edit.gif" alt="loginpage" style={{ width: '350px', height: 'auto',boxShadow: '0 0 10px 0 rgba(0, 0, 0, 0.1)' }} />

## AI生成内容
- **操作**：配置AI的token后，点击右下角的AI生成按钮或使用键盘 `Ctrl+J` 生成内容。
- **功能**：自动生成笔记内容，辅助用户快速创建内容。
<img src="/screenshot/AI_demo.gif" alt="loginpage" style={{ width: '350px', height: 'auto',boxShadow: '0 0 10px 0 rgba(0, 0, 0, 0.1)' }} />


## 添加标签
- **操作**：在笔记中输入 `#tag` 进行标签添加。
- **功能**：方便后续按标签分类和搜索笔记。
<img src="/screenshot/tag_demo.gif" alt="loginpage" style={{ width: '350px', height: 'auto',boxShadow: '0 0 10px 0 rgba(0, 0, 0, 0.1)' }} />

## 归档笔记
- **操作**：点击界面上的归档按钮。
- **功能**：将当前笔记移动到归档文件夹，不再显示在主界面。
<img src="/screenshot/archive_demo.gif" alt="loginpage" style={{ width: '350px', height: 'auto',boxShadow: '0 0 10px 0 rgba(0, 0, 0, 0.1)' }} />

## 浏览与搜索已归档笔记
- **操作**：点击右上角的浏览按钮。
- **功能**：打开已归档笔记的浏览界面，支持按时间排序和关键词搜索（关键词用空格分割）。
<img src="/screenshot/search_demo.gif" alt="loginpage" style={{ width: '350px', height: 'auto',boxShadow: '0 0 10px 0 rgba(0, 0, 0, 0.1)' }} />

## 为无标签的归档笔记添加标签
- **操作**：在归档笔记浏览界面中，选择无标签的笔记进行标签分类。
- **功能**：帮助整理和分类归档笔记，便于后续查找。
